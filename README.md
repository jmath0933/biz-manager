# 💼 Biz Manager

**Biz Manager**는 거래처 관리, 매입/매출 관리, 통계 분석을 한 곳에서 할 수 있는 웹 기반 비즈니스 관리 시스템입니다.  
Next.js 16 (App Router), TypeScript, Tailwind CSS로 구축되었습니다.

---

## 🚀 주요 기능

### 🔐 로그인
- 사용자 계정 로그인 페이지  
- 로그인 성공 시 `대시보드`로 이동  
- 세션 기반 로그인 상태 유지  

### 📊 대시보드
- 메인 홈에서 주요 기능 버튼 제공  
  - 거래처 관리  
  - 매입 관리  
  - 매출 관리  
  - 통계 보기  

### 🏢 거래처 관리
- 거래처명, 대표자명, 사업자등록번호, 거래은행, 계좌번호, 전화번호, 이메일, 비고 필드 관리  
- 전화번호 자동 하이픈 입력  
- 통화 버튼 (모바일 클릭 시 전화 연결)  
- 이메일 버튼 (Gmail/Naver 메일 클라이언트 자동 연결)  
- 거래은행 선택 시 은행별 계좌번호 형식 자동 적용  

### 💸 매입/매출 관리 (준비 중)
- 거래처별 매입/매출 내역 등록  
- 합계, 미수금, 미지급금 자동 계산  

### 📈 통계
- 월별 매입/매출 그래프 표시 (Recharts 예정)  
- 거래처별 비율 분석
